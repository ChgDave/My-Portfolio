Art Block Breaker

The art block breaker is my first coding bootcamp group project. For this project we have to research open source APIs and build a website concept around them. We are required to use at least two different APIs, CSS framworks, incorporate user inputs and save information to local storage and most importantly collaborate with our teammates to finish this projct.

Our concept is to create an art exploration and idea generation tool. By using APIs from the Art Institute, the user is able to discover pieces of art they enjoy and save specific pieces to a favorites list. The user is then able to generate a word cloud using a word cloud generator API to provide inspiration for their own art.
