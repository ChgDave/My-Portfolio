Weather Dashboard

The weather dashboard allows you to search for the weather forecast for the next 5 days of a city.

This app uses the openweathermap API to return the geo location of the city your choose, then uses the geo location to retrive the weather forecast data of that city. The searched result will also be stored in localstorage so that the data persist and will be available every time the web page loads.
