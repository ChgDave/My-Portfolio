Pig Game

The pig game is a fun little game that I developed. For this game, you can play with a partner to roll the dice and bank your score. The player who first reaches 100 points wins the game.

This game is a great way to spend some time with a friend. It is an interactive game and you need to make the decision on when to bank the score so it adds some strategy as well.
